# useful

## Installation

$ `curl https://raw.githubusercontent.com/claromes/useful/main/useful.sh --output useful.sh`

$ `sudo chmod +x ./useful.sh`

In the `.bashrc` file, add the following line:

```bash
alias useful='./useful.sh'
```

## Usage

$ `useful utils`

$ `useful docker`

$ `useful git`

$ `useful python`
